# PID 0x4C: Front HVAC Main Panel

| Byte | Function | Logic / Values |
| :--- | :--- | :--- |
| **0** | **Auto Button** | Bit 0 (`& 0x01`) |
| **1** | **Rear Control** | High Nibble is `0xF0` |
| **2** | **Fan Speed** | Up: `0x23/0x34`, Down: `0x43/0x32` |
| **3** | **Temp Left** | Up: `0x0D/0x0F`, Down: `0x07/0x09` |
| **4** | **Temp Right** | Up: `0x0D/0x0F`, Down: `0x07/0x09` |
| **5** | **Air Direction L** | Up: `0x65/0x86`, Down: `0x68/0x56` |
