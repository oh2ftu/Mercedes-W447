# PID 0x4C - Front HVAC Control Panel

This PID handles the physical button interactions for the front climate control unit. Rocker switches are mapped to a consolidated state sensor.

## Status Mapping

| Byte | Function | Idle | Up / On | Down |
| :--- | :--- | :--- | :--- | :--- |
| **0** | Auto Button | `0x10` | `0x11` (Bit 0) | - |
| **1** | Rear Mode Active | `0xE0` | `0xF0` (Bit 4) | - |
| **2** | Fan Speed Rocker | `0x55` | `0x56` / `0x66` | `0x65` |
| **3** | Left Temp Rocker | `0x0B` | `0x0D` / `0x0F` | `0x09` / `0x07` |
| **4** | Right Temp Rocker | `0x0B` | `0x0D` / `0x0F` | `0x09` / `0x07` |
| **5** | Air Direction Rocker| `0x55` | `0x65` / `0x66` | `0x56` |

## Implementation Notes
To prevent "sticky" states in Home Assistant, the logic must explicitly check for the `Idle` hex values to reset the template sensor.
