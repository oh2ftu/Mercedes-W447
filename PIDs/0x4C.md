## PID 0x4C: Master Climate Control (Request)
**Length:** 8 Bytes
**Frequency:** ~100ms / Event-based

| Byte | Bit(s) | Function | Values / Notes |
| :--- | :--- | :--- | :--- |
| 0 | - | Message State | `10`=Standard, `14`=Active/Rapid Update |
| 1 | - | Air Direction | `E0`=Face, `E2`=Feet, `F0`=Defrost/Mix |
| 2 | - | Fan Speed | `00` (Off), `11`, `22`, `33`, `44`, `55`, `66`, `77` (Max) |
| 3 | - | Driver Temp | `00`=LO, `03`=16°C ... `09`=20°C ... `1E`=HI |
| 4 | - | Passenger Temp| Matches Driver Temp scale |
| 5 | - | HVAC Mode | `55`=Normal, `99`=MAX A/C / Defrost |
| 6 | - | Unknown | Const `3F` |
| 7 | - | Interaction | `09`/`11`=Button Active, `01`=Idle |

## PID 0x8B: HVAC Status / LED Feedback
**Note:** Byte 1 mirrors 0x4C Byte 2 (Fan Speed).
**Marker Bit:** Byte 0, Bit 4 toggles the Interior Circulation LED.
