## PID 0x0D: Rear HVAC Control (Request)
**Logic:** State-based (set values), mirrors 0x4C logic.

| Byte | Function | Values / Notes |
| :--- | :--- | :--- |
| 0 | Message State | `10`=Idle, `14`=Changing |
| 1 | Air Direction | `E0`=Standard Face |
| 2 | Fan Speed | `00` (Off) to `77` (Max) - Matches 0x4C scale |
| 3 | Rear Temp L | Target Temp + `0x60` (e.g., `69` = 20°C, `7E` = HI) |
| 4 | Rear Temp R | Target Temp + `0x60` |
| 5 | Mode Flags | `88`=Standard, `66`=Transition/Alternate |
