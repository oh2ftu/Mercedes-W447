# PID 0x0D - Rear HVAC Control Panel (P2)

This PID handles the button interactions for the rear passenger climate controls.

## Status Mapping

| Byte | Function | Idle | Up | Down |
| :--- | :--- | :--- | :--- | :--- |
| **2** | Rear Fan Speed | `0x66` | `0x77` | `0x76` |
| **3** | Rear Temp Rocker | `0x6B` | `0x6D` / `0x6F` | `0x69` / `0x67` |

## Observation
Note that the neutral (Idle) offsets for the rear panel differ from the front panel (e.g., `0x66` vs `0x55`).
