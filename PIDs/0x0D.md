# PID 0x0D: Rear HVAC Control Panel

**Description:** Captures inputs from the rear passenger climate unit.

| Byte | Value | Function |
| :--- | :--- | :--- |
| 0 | - | Unknown / Research Needed |
| 1 | - | Unknown / Research Needed |
| **2** | `0x01` | **Rear Fan Up** |
| **2** | `0x11` / `0x12` | **Rear Fan Down** |
| **2** | `0x00` | Rear Fan Neutral |
| **3** | `0x6D` / `0x6F` | **Rear Temp Up** |
| **3** | `0x69` / `0x67` | **Rear Temp Down** |
| 4-7 | - | Unknown |

> **Note:** The Rear Fan uses a specific toggle logic (`0x11` vs `0x12`) depending on the speed of the press or current state.
