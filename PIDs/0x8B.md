# PID 0x8B: Indicator LEDs

These bits control the physical lights on the button panels.

| Byte | Bit | Function | Logic |
| :--- | :--- | :--- | :--- |
| **0** | **5** | **Menu LED** | `0x20` Mask |
| **0** | **4** | **Int Circ LED** | `0x10` Mask |
| **0** | **2** | **Auto LED** | `0x04` Mask |
| **5** | **5** | **Rear Heat LED** | `0x20` Mask |
| **5** | **3** | **Demist LED** | `0x08` Mask |
| **5** | **2** | **Aux Timer LED** | `0x04` Mask |
| **5** | **1** | **Aux Vent LED** | `0x02` Mask |
| **5** | **0** | **Aux On LED** | `0x01` Mask |
