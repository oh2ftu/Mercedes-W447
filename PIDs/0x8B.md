# PID 0x8B - HVAC LED Indicators

This PID contains the status of the physical LEDs on the control buttons.

## Bitmask Table

| Byte | Bit | Mask | Function | Note |
| :--- | :--- | :--- | :--- | :--- |
| **0** | 5 | `0x20` | **Rest LED** | Residual Heat active |
| **0** | 4 | `0x10` | **Int Circ LED** | Air recirculation active |
| **0** | 2 | `0x04` | **Auto LED** | Auto mode active |
| **2** | 1 | `0x02` | **Rear Ctrl LED** | Active when front controls rear (~2s) |
| **5** | 5 | `0x20` | **Rear Heat LED** | Rear window defroster |
| **5** | 4 | `0x??` | **SYNC LED** | AC Sync |
| **5** | 3 | `0x08` | **Demist LED** | Max windshield defrost |
| **5** | 2 | `0x04` | **Aux Timer LED** | STH schedule active |
| **5** | 1 | `0x02` | **Aux Vent LED** | STH in ventilation mode |
| **5** | 0 | `0x01` | **Aux On LED** | STH active |
